SVG

SVG or Scalable Vector Graphics is a World Wide Web Consortium (W3C) standard.
It is used for the creation and display of vector graphic material.
The first public draft of SVG was released by the World Wide Web consortium in February of 1999.

Browser support

Advantages of SVG

  • Much smaller than bitmaps
  • Can be scaled without the pixelation
  • Reducing the server load and network response time
  • The end-user can interact with the graphics without costly client-server communications
  • Native support for SMIL (Synchronized Media Integration Language)
  • It responds to JavaScript

Coordinate system
